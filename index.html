ACIDUINO

8USD/ USD equivalent for other currency

ACIDUINO is a clone of TB-303's sequencer engine using Arduino. It is a PCB version of the schematic designed by Midilab.
Currently, we have two types in stock, black and purple.

With little assembly and a few simple builds, you're ready for some ACID phrases!

Please refer to the repository below for more details.
Midilab's ACIDUINO PCB repositories
We are grateful to Midilab for their permission to sell it.

HAGIWO CHORD VCO

10USD/ USD equivalent for other currency

Arduino SamuraiHAGIWO designed CHORDVCO and I converted it to PCB.
It requires Arduino to be mounted, but it is easy to assemble with few standard parts.
The output is designed with Vp-p=4V, so you may feel that the volume is lower than other modules on the market. This is as designed by HAGIWO.

Please refer to my repository for BOM and other information.
The Arduino code can be found on the HAGIWO website.

HAGIWO's V2044 VCF

10USD/ USD equivalent for other currency

This is a PCB version of the V2044 VCF designed by HAGIWO. This VCF uses the same 2044 IC used in KORG PLYSIX.A simple but effective filter. The circuit diagram is based on HAGIWO's original. I actually manufactured it with JLCPCB and assembled it, and it works.

！！ Note ！！


For the v2044 IC chip itself, I used the Coolaudio reissue version.
The v2044 package is SOP16, and only this part requires surface mount technology. (Might be easier than you think!)
See HAGIWO's original schematic for details.
I will get feedback and improve it.
Please see my repository for BOM and other information.
